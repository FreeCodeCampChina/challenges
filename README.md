# branch
master 分支存放的是最新的英文课程，此分支会有专人来更新。

middle 分支存放的是稳定的英文课程，此分支间会有专人 compare master 分支后再更新。

translate 分支是仓库的默认分支，也是翻译者和审批者唯一需要关注的分支。
